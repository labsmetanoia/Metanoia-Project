# 07 · Features & Roadmap

Sequencing principle (from the founding blueprint): *close the gap in order — prototype → Forage-level → world-class — never everything at once.* Each phase must produce its own proof before the next unlocks funding, partners, and data.

## 1. Phase Map

| Phase | Name | Duration | Theme | Proof it must produce |
|---|---|---|---|---|
| **0** | Foundation | 6–8 wks | Identity, diagnostic, landing | Waitlist → activated users; diagnostic completion >70% |
| **1** | MVP — "The Map" | 3–4 mo | Compass + core Forge + light Circle | Weekly retention >35%; readiness lift measurable; NPS >50 |
| **2** | "The Bridge" | +4–6 mo | Employers, simulations, credentials, mentors | 5–10 paying employer partners; first attributed placements |
| **3** | "The Flywheel" | +6–9 mo | Alumni regeneration, marketplace, educator portal, mobile app | Alumni-mentor conversion >15%; university cohorts live |
| **4** | "The Infrastructure" | year 3+ | Atlas at scale, B2G/institutional data products, multi-region | Institutional contracts; recognized credential standard |

## 2. MVP Feature Set (Phases 0–1)

The MVP is **the transformation journey for one user segment** (final-year students & fresh grads targeting business/tech roles in Indonesia), not a thin slice of every engine.

### Must-have (launch-blocking)

| Engine | Feature | Notes |
|---|---|---|
| Identity | Auth (email + Google/LinkedIn), Passport v1 schema | The sacred schema — built for the 5-year model |
| Compass | 15-question Readiness Diagnostic + gap-map reveal | The free, generous front door |
| Compass | AI 90-day Pathway (generated, weekly re-plan) | Core intelligence product |
| Compass | Meta coach v1 (journey-aware chat, next-best-action) | Scoped: navigation + reframing + routing |
| Forge | CV Studio with ATS scoring vs. target JD + fix-list | Day-1 magic moment: score lift in first session |
| Forge | Interview Coach v1 (text/audio mocks, STAR feedback, pattern tracking) | |
| Circle | Cohorts (auto-placed, ~20 users) + feed | Community from day 1, but small and warm |
| Circle | Peer matching + guided mock-interview sessions | The reciprocity mechanic |
| Atlas | Intelligence Library v1 (30–50 insider guides, EN/ID) | Also the SEO engine |
| Platform | Readiness Ring + Path UI, notifications (email/in-app), mobile-responsive web | |
| Ops | Admin basics: user management, content CMS, moderation, analytics events from day 1 | Event schema designed before launch |

### Explicitly deferred from MVP
Simulations marketplace, employer portal, paid tiers, mentor marketplace (founder-led manual mentoring instead), native apps, assessment prep banks, case practice, offer-negotiation tools, sub-communities.

## 3. Phase 2 — "The Bridge" (employer & credential layer)

- **Employer Portal v1:** simulation builder (with heavy Metanoia co-creation services at first), pipeline dashboard, job postings.
- **Simulations engine:** multi-task player, AI+rubric scoring, verified certificates → Passport credentials.
- **Talent discovery (opt-in):** consent-gated candidate signals, contact requests.
- **Mentorship Marketplace v1:** alumni + recruited professionals; free community slots + paid sessions (payments: Xendit/Stripe).
- **Premium individual tier:** unlimited AI depth, advanced mocks, priority mentors.
- **Application Command Center**, assessment prep banks, resilience programming v1.

## 4. Phase 3 — "The Flywheel"

- **Alumni regeneration system:** placement detection → mentor rite-of-passage onboarding → contribution reputation.
- **Educator Portal:** cohort tooling, outcome dashboards, SAML SSO, curriculum embedding.
- **React Native app** (mobile-first market), offline-tolerant content.
- **Case & presentation practice**, sub-communities & AMA events.
- **Atlas v2:** live market intelligence feed, readiness prediction from real outcome data (replacing heuristics).
- **Public impact dashboard** (transparent outcomes).

## 5. Phase 4 — "The Infrastructure"

- Atlas institutional products (aggregate workforce intelligence for government/universities/employers).
- Verifiable credential standard (Open Badges/VC alignment; registry integrations).
- Multi-language/multi-region (SEA expansion), government talent-system and university-ERP integrations.
- Enterprise-grade SLAs, SOC 2, multi-region infra (per doc 08).

## 6. AI Feature Ladder

| Tier | Capability | Phase |
|---|---|---|
| 1 | Diagnostic gap analysis; pathway generation; ATS scoring; STAR feedback | MVP |
| 2 | Journey-memory coaching (Meta with full context); cross-session pattern detection; content personalization | MVP–2 |
| 3 | Simulation auto-assessment; mentee briefs for mentors; employer match signals (embeddings) | 2 |
| 4 | Readiness prediction from outcome data; dynamic market-driven re-planning; cohort risk flags | 3 |
| 5 | Labor-market intelligence models; skill-velocity forecasting; institutional analytics | 4 |

Guardrails at every tier: explainability, no fabricated stats, distress → human escalation, cost routing (small models for volume, frontier for high-stakes).

## 7. Behavioral Engagement Systems

Design target: **weekly active journey participation**, not daily app opens.

| System | Mechanic | Phase |
|---|---|---|
| Weekly intention → reflection loop | Monday: pick 3 actions; Sunday: reflect + readiness delta | MVP |
| Next-best-action engine | One hero action on Basecamp, always | MVP |
| Cohort social proof | Wins celebrated in-feed; milestone moments | MVP |
| Reciprocity credits | Give a mock, get a mock — session economy | MVP |
| Contribution reputation | Visible generosity status; mentor pipeline | 2–3 |
| Comeback flows | Stall detection (10 days idle) → compassionate re-entry, not guilt push | 2 |
| Milestone rites | First mock, first interview, offer, mentor induction | MVP–3 |

Anti-patterns banned: punitive streaks, engagement-bait notifications, leaderboards of people (leaderboards of *generosity* only), infinite feeds.

## 8. Personalization Systems

1. **Profile-based:** target role, background, timeline, hours/week → pathway shape.
2. **Behavioral:** practice performance & patterns → next actions, content ordering.
3. **Market-based:** Atlas signals → pathway re-weighting, insight cards.
4. **Community-based:** stage/goal/city matching for peers, cohorts, mentors.
5. **Emotional:** rejection events & sentiment cues → tone shifts, resilience content, human routing.

## 9. Feature Prioritization Rubric

Score 1–5 each; build ≥16 first: **Access-gap impact** (does it give the un-networked what the networked have?) · **Outcome proximity** (does it move interviews/offers?) · **Flywheel contribution** (does it feed regeneration/employer/data loops?) · **Effort inverse** · **Proof value** (does it unlock the next phase's partners/funding?).
